- Login page
- Home Page

## Tech Stack
-   React.js
-   Redux
-   JavaScript
-   chakra-ui

## Tech Stack
-   Node.js
-   Express.js
-   MongoDB  
-   Mongoose
-   Jwt
-   Bcrypt

  # API Endpoint
- `POST `    /user/register
- `POST`     /user/login
- `PUT`      /user/:id  

- `GET  `    /category
- `GET`      /category/:id
- `POST`     /category/
- `DELETE`   /category/:id
- `PATCH`    /category/:id

- `GET  `    /product
- `GET`      /product/:id
- `POST`     /product/
- `DELETE`   /product/:id
- `PATCH`    /product/:id
